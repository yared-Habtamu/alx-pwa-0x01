# alx-movie-app (Cine Seek)

This minimal scaffold wires up `@ducanh2912/next-pwa` and `webpack` so the app is PWA-ready.

Verification

- Check `package.json` for these entries:

```json
"@ducanh2912/next-pwa": "^10.2.9",
"webpack": "^5.94.0"
```

Run locally

```pwsh
cd alx-movie-app
npm install
npm run dev
```
